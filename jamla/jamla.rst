Jamla
================

----------------------------------------
Just a Markup Language Application
----------------------------------------

Jamla is a jaml file which describes all information needed to build your 
subscription based website.

:ref:`jamla-latest` shows the latest definition of a valid Jamla file.

A jamla.yaml file is a `standard <https://xkcd.com/927/>`_ way to describe your
website's:

* Products & their unique selling points
* Each products pricing rules
* Settings for the application
